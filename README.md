# NexaFlow

A premium SaaS landing page for an AI-driven data automation platform. Built with Next.js, TypeScript, and Tailwind CSS.

## Features

- Responsive hero, feature showcase, pricing matrix, and social proof sections
- Matrix-driven multi-currency pricing (INR, USD, EUR) with monthly/annual billing
- Desktop bento grid that reflows into a mobile accordion with context preservation on resize
- SEO-optimized with semantic HTML, Open Graph metadata, sitemap, and robots.txt
- CSS and WAAPI animations with performance-isolated pricing state updates

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS v4

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3001](http://localhost:3001).

```bash
npm run build
npm run start
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start development server on port 3001 |
| `npm run dev:clean` | Free port 3001 and start dev server |
| `npm run build` | Production build |
| `npm run start` | Start production server |
